Artefacto: pieza de informacion modificada o creada en un proceso
ito: objetivo que quiero que se cumpla en determinado tiempo

Punto de partida es de que se trata el dominio. ej una empresa nueva tiene
distinto punto de partida que una que trabaja hace 10 años o una que
tiene un sistema de gestion obsoleto. Es decir que empiezo de forma 
distinta para cada caso

Dominio es un area de conocimiento, tiene su propia semantica
ej para una farmacia la semantica son clientes, laboratorios, medicamentos,
recetas, etc

Una herramienta es mas que nada draw.io y demas para construir todas las
cosas

Clase es la representacion de un conjunto de objetos que tiene la 
particularidad de compartir los mismos atributos, operaciones, relaciones y 
semantica. Es para categorizar y representar los objetos
Objeto: Instancia de una clase

Diagrama de clases semantico permite reconocer los objetos del dominio y
representarlos en clases. Participan las clases de entidad

Diagrama de clases como colaboracion simple tiene las clases que colaboran 
en un caso de uso. Se llama realizacion

Diagrama de clases como esquema logico de base de datos es el punto de 
partida para el diseño de la base de datos
